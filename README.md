# OrangePi MPI3501
> **Work in Progress**
>
> Bringing native MPI3501 (ILI9486 + XPT2046) support to Orange Pi H3 boards running modern Armbian.

ILI9486 + XPT2046 display support for Orange Pi H3 boards.
## Supported hardware

### SBC

- [x] Orange Pi PC (H3)

### LCD

- [x] MPI3501
- [x] ILI9486
- [x] XPT2046

Tested on:

- Orange Pi PC
- Armbian 26.x
- Linux 6.18
- KlipperScreen

Features

- SPI display
- Touchscreen
- Device Tree Overlay
- One-command installer

Status

🚧 Under development
