# Contributing

Thank you for your interest in this project.

## Project Philosophy

- Test on real hardware.
- Verify before assuming.
- Prefer upstream Linux solutions over custom hacks.
- Document every important finding.
- Keep the project simple and reproducible.

## Development Rules

- One feature per commit.
- Test before merging.
- Update documentation when behavior changes.
- Keep compatibility with modern Armbian kernels whenever possible.

## Hardware

Current development platform:

- Orange Pi PC (H3)
- Armbian 26.x
- Linux 6.18.x
- MPI3501 (ILI9486 + XPT2046)
